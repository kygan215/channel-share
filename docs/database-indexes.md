# 云数据库索引配置指南

> 在微信开发者工具 → 云开发 → 数据库 中配置以下索引

## 1. `channels` 集合

| 索引名称 | 字段 | 方向 | 说明 |
|---------|------|------|------|
| `idx_id` | `id` | 升序 | 按数字ID查询单条记录（getDoc） |
| `idx_province` | `province` | 升序 | 按省份筛选过滤 |
| `idx_phone` | `phone` | 升序 | 按手机号去重/查询 |
| `idx_hired` | `hired` | 升序 | 按聘用状态筛选 |
| `idx_creatorUserId` | `creatorUserId` | 升序 | 按创建者查询（编辑/删除权限校验） |
| `idx_export` | `province` + `serialNo` | 升序 + 升序 | 导出Excel排序用 |

## 2. `users` 集合

| 索引名称 | 字段 | 方向 | 说明 |
|---------|------|------|------|
| `idx_openid` | `openid` | 升序 | 按openid查找用户（登录鉴权） |
| `idx_role` | `role` | 升序 | 按角色筛选管理员 |

## 3. `whitelist` 集合

| 索引名称 | 字段 | 方向 | 说明 |
|---------|------|------|------|
| `idx_phone` | `phone` | 升序 | 按手机号验证白名单 |

## 4. `whitelist_users` 集合

| 索引名称 | 字段 | 方向 | 说明 |
|---------|------|------|------|
| `idx_openid` | `openid` | 升序 | 按openid查询用户白名单关联 |
| `idx_phone` | `phone` | 升序 | 按手机号查询关联 |

## 配置步骤

1. 打开微信开发者工具
2. 点击左上角「云开发」按钮
3. 在弹出窗口中选择「数据库」标签
4. 依次选择每个集合，点击「索引」选项卡
5. 点击「添加索引」，按上表填入字段名和方向
6. 点击「确定」保存

> ⚠️ 注意：索引创建后需要几分钟生效。建议在上线前配置好。
